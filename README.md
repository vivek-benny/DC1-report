# Overview

This repository documents a complete penetration testing assessment of the **DC-1 VulnHub machine** conducted in a controlled virtual lab.

The objective was to identify vulnerabilities, exploit exposed services, escalate privileges, and obtain root access.

The attack lifecycle followed:

Reconnaissance → Enumeration → Exploitation → Post-Exploitation → Privilege Escalation → Flag Retrieval

---

# Lab Environment

Attacker Machine  
Kali Linux

Target Machine  
DC-1 VulnHub (Debian Based Linux)

Virtualization  
Oracle VirtualBox (Host-Only Network)

Tools Used

- Nmap
- Metasploit Framework
- Linux Enumeration Commands
- Python PTY shell stabilization
